# spaCy Annotation Converter
This tool is needed to convert the json annotation data exported form this [online annotator](https://tecoholic.github.io/ner-annotator/).
### Part of the code is taken from [amrss/custom-ner-with-spacy3 repo](https://github.com/amrrs/custom-ner-with-spacy3/blob/main/Custom_NER_with_Spacy3.ipynb)
## How to use
Once downloaded the json annotation data you can run the following script and you will get the training dataset in .spacy format.

`python spaCy-Json-Converter -i <your_file_path> -o <your_output_path>`
